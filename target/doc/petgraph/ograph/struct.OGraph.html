OGraph<N, E> is a directed graph using an adjacency list representation.

The graph maintains unique indices for nodes and edges, so both node and edge data may be accessed mutably.

Based upon the graph implementation in rustc.

NodeIndex and EdgeIndex are types that act as references to nodes and edges, but these are only stable across certain operations. Adding to the graph keeps all indices stable, but removing a node will force another node to shift its index.

Removing an edge also shifts the index of another edge.
